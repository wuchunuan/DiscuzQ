## Installing:
In your pubspec.yaml
```yaml
dependencies:
  status_alert: ^0.1.2
```
```dart
import 'package:status_alert/status_alert.dart';
```


## Basic Usage:
```dart
    StatusAlert.show(
      context,
      duration: Duration(seconds: 2),
      title: 'Title',
      subtitle: 'Subtitle',
      configuration: IconConfiguration(icon: Icons.done),
    )
```
